Welcome

FlexBuilder is a lovely piece of work, no denying it. But have you every used the Eclipse Java environement? Now that's a lovely piece of work. It's what all IDE's should aspire to, FlexBuilder's on the way but it's not quite there yet.

The Flex Tools project aims to give FlexBuilder (and in turn, you, the developer) a helping hand in increasing your productivity.

Quick start

Raring to go? Here's what you need to do:

  1. Download Eclipse 3.2
  2. Get the plug-in version of FlexBuilder
  3. Use the update manager to access the remote update site at http://flextools.berlios.de/update/
